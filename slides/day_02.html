Control flows

  • Refering to changing the flow of the program/script
  • So far: Python executes line by line, from top to bottom
  • We can not do any logic:
    • Optional execution of statements (If/Then branch)
    • Execute these statements instead of those (If/Else branch)
    • Repeat these statements (Loop)

Code block

  • A bunch of statements performing one logic operation is called a code block
  • Statements of the same blocks are grouped together using indentation
  • A code block can contains other code blocks
    • 
code_block_0_statement1
code_block_0_statement2
    codeblock_1_statement1
    codeblock_2_statement2
    codeblock_3_statement3
code_block_0_statement3
code_block_0_statement4

Indentation

  • You can use tab or multiple spaces for indentation
  • Convention: Use 4 spaces
  • You should be consistent: If you use one, stick with one for the entire script
  • Using an IDE like PyCharm helps convert indentation to one style (which is configurable)
  • You can safely use the tab key for all indentation, PyCharm will convert to 4 spaces

If/Then branch

Syntax: 

statement0
statement1
if condition:
    statement2
    statement3
statement4
  • Solve this problem: Optional execution of a code block
  • Does not change the script's normal behavior
  • But do some additional things

print("Don't forget your phone")
print("Don't forget your keys")
if temperature < 70:
    print("Don't forget your jacket either!")
    print("Don't forget your lunch")  # Is this what we want?
print("Don't forget to check mail")

If/Else branch

Syntax: 

statement0
if condition:
    statement1
    statement2
else:
    statement4
    statement5
statement6
  • Solve this problem: Execute either this block or that block, but not both
  • This does change the script behavior
  • Script has to take one of the code path depending on the condition

if temperature < 70:
    print("It's cold out there")
else:
    print("It's hot out there")
print("Dress properly!"")

If/Then vs If/Else

  • We can convert If/Then branch to If/Else branch by using empty block
  • To signal an empty block, use the keyword pass in the empty block
  • The following two examples do exactly the same thing:

if temperature < 70:
    print("It's cold")
                    

if temperature < 70:
    print("It's cold")
else:
    pass

Branches example #2

Write a script to ask for current user's age, and print out the age. Remind him/her to vote if he/she is 18 or older, else print an error

What is the difference?
  • Write a script to ask for current user's age, print out the age, and remind him/her to vote if he/she is 18 or older
  • Write a script to ask for current user's age, and print out the age. Remind him/her to vote if he/she is 18 or older, else print an error
What is the difference?

answer = input('How old are you? ')     # 1. Get current users input

if age >= 18:                           # 4. Remind user to register

else:                                   # Else print an error

print('Done')

Loop

  • Repeat these statements until condition
  • Python syntax: Use while not condition: or while exit_condition:
  • exit_condition is the reverse of until condition
  • If a variable is used within exit condition, it must be initialized to some value
  • The variable must be updated within the loop
  • Example 1: print 'A', print 'B', print 'C', until answer == 'Yes!'
  • Translate to python: 
    
answer = input('Do you like Python? ')
while not (answer == 'Yes'):
    print('Python is wonderful!')
    print('Python is awesome!')
    print('You should like Python!')
print('Done')
  • !!! ERROR !!! Your script never finishes
  • answer is asked once, but is not updated
  • The code block continues to run, infinitely: infinite loop
  • Fix:

answer = input('Do you like Python? ')
while not (answer == 'Yes'):
    print('Python is wonderful!')
    print('Python is awesome!')
    print('You should like Python!')
    answer = input('Do you like Python? ')  # This line is here to update answer
print('Done')

Loop example

Write a script to ask a user to enter a number larger than 10, then start counting upto that number

  • Get user's input as end_number, and convert that to an int
  • Initialize a count to 1
  • Start counting, until count is upto user's input
  • Update count so we do not have an infinite loop

answer = input('Please enter a number larger than 10:')
end_number = int(answer)
count = 1
while not (count == end_number + 1):   # Why is it end number + 1???
    print(count)
    count = count + 1
print('done')

Loop (cont.)

  • A loop continues to execute, unil the exit condition is True
  • We can 'prematurely' exit the loop using break
  • We can 'prematurely' restart the loop using continue
Base script

num = 0
while num < 10:
    print('The number is', num)
    num = num + 1 # increment num, using the current value
    print('The next number is', num)
print('done')
What if we add a break

num = 0
while num < 10:
    print('The number is', num)
    num = num + 1 # increment num, using the current value
    break
    print('The next number is', num)
print('done')
What if we add continue

num = 0
while num < 10:
    print('The number is', num)
    num = num + 1 # increment num, using the current value
    continue
    print('The next number is', num)
print('done')

Count from 0 to 100, but stop after a number specified by the user

  • Get user's answer, convert to an int, and store in a variable named end
  • Initialize the counter to 0
  • Start counting, until counter is up to 100
  • Within the loop: Update the counter by incrementing
  • Within the loop: Check if counter > end then break

answer = input('Enter the stop number: ')
end = int(answer)
counter = 0
while not (counter > 100):
    print('Counting number:', counter)
    counter = counter + 1
    if counter > end:
        print('I am stopping after', end)
        break
print('done')

answer = input("What multiplication table number do you like to see? ")

if number < 1:
    print("Error: Number must be greater or equal to 1")
elif number > 10:
    print("Error: Number must be less than 10")

Assignment #4: FizzBuzz!

  • Write a script to ask the user for a number between 20 and 100
  • Count to that number. However...
  • If the current number is divisible by 3, print "Fizz" instead
  • If the current number is divisible by 5, print "Buzz" instead
  • If the current number is divisible by both 3 and 5, print "FizzBuzz"
Assignment #4: FizzBuzz!
  • What's our input? number
  • What kind of processing/logic do we need?
    • Validate the number is between 20 and 100, using...If/Elif
    • Initialize counter = 1
    • Loop the following until counter == number
      • What kind of branch do we use? If/Elif
      • if counter % 3 == 0: print Fizz
      • elif counter % 5 == 0: print Buzz
      • elif counter % 15 == 0: print FizzBuzz
      • else print counter
number = int(input("Please enter a number between 20 and 100: "))
if number < 20:
    print("Error! Number is too small")
elif number > 100:
    print("Error! Number is too large")
else:
    counter = 1
    while not (counter == number + 1):
        if (counter % 3) == 0:
            print("Fizz")
        elif (counter % 5) == 0:
            print("Buzz")
        elif (counter % 15) == 0:
            print("FizzBuzz")
        else:
            print(counter)
        counter = counter + 1        # Else, infinite loop!

What's wrong?

  • We expect 15 to be replaced with FizzBuzz
  • But we only see Fizz. What branch execute when counter is 15?
  • if (counter % 3) == 0: This branch executed
  • Problem: The cases are not mutually exclusive
  • Divisible by 15 also means divisible by 3 and 5
  • But the reverse is not True
  • Solution: Check for the "larger" condition first!
number = int(input("Please enter a number between 20 and 100: "))
if number < 20:
    print("Error! Number is too small")
elif number > 100:
    print("Error! Number is too large")
else:
    counter = 1
    while not (counter == number + 1):

        if (counter % 15) == 0:
            print("FizzBuzz")
        elif (counter % 3) == 0:
            print("Fizz")
        elif (counter % 5) == 0:
            print("Buzz")
        else:
            print(counter)
    counter = counter + 1        # Else, infinite loop!

Lession learned from FizzBuzz

  • Watch out for conditions that are not mutually exclusive
  • Sometime, the coding logic is different from the wording of the prolems
