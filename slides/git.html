Git

Source code control

  • Helps organize and track source code, or text files
  • Keep a good history to document all changes
  • Sync changes amongst multiple contributors
  • The new standard: git

GIT

  • By itself, git is an open source project
  • Download git 64bit using the following URL
  • Install Git, accepting all the default settings
  • If you are on Linux, use your own package manager for installing git
  • Let's start a new git project using PyCharm

Github

  • You have a project directory locally
  • How do you share it with people?
  • And, how do you contribute to another project?
  • Introducint github

  • Github host your project for you
  • You have to push your project to github
  • You promise to be nice: Open source your project
  • Github also have:
    • Issues/ticket tracking system
    • Statistics and pretty graphs
    • Collaboration tools

Using Github

Git cmd

  • You do not need PyCharm, or GUI for git
  • Everything can be done via the command line

git clone https://github.com/username/someproject
cd someproject
git log
(edit some files)
git status
git add file1
git add file2
git commit -m "Some message"
git push

  • GUI is nicer and easier to use
  • But...most tutorial uses the command line
  • You should be familiar with the command line to follow tutorial
  • git is hard
  • I mess up all the time
  • Don't feel bad. It takes time to have a "feel" for git
